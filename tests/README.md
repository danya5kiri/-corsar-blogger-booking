# Проверка здоровья сайта «Корсар»

Файл `site_health_test.js` запускается автоматически через GitHub Actions при каждом pull request и изменении ветки `main`.

Проверяются:

- синтаксис основного JavaScript и уникальность статических HTML-ID;
- получение прогноза Open-Meteo Best Match на 16 дней и отображение данных при переходе между месяцами;
- границы сезона и количество уникальных категорий туров;
- занятость одного тура на конкретную дату;
- создание, перенос и отмена бронирования;
- запрет управления прошедшими или чужими бронями;
- соответствие выбранного тура расписанию выбранной даты;
- рекомендация только свободной и ещё не посещённой категории;
- подтверждение операций после повторной загрузки данных;
- обезличивание записей локального журнала диагностики.

Тесты не отправляют заявки в рабочую Google-таблицу и не используют реальные персональные данные.
